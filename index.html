Java + SQL Project

Visit GitHub


This GitHub repository contains E-commercial management system applications using Oracle PL/SQL and Java. The purpose of this project is to generate more profits by selling goods to a wider customer base through E-commercial.
Based on the Oracle SQL Developer program, tables based on the characteristics of the information contained in PL/SQL have been created. The total number of the table is six, which are 'Product', 'Customers', 'Category', 'OrderDetail', 'Orders', and 'Shipping' in the Entity-Relationship Diagram(ERD).
In this project, JavaFX is used to run consistently across multiple platforms and various devices such as desktop, mobile phones, tablets, TV, and etc.

What is JavaFX?

JavaFX is a Java library that allows users to build Rich Internet Applications. Moreover, JavaFX also provides interfaces using which developers can combine graphics, animation and UI control. For more details, please refer to the following link: See Details

Microsoft Excel
Data Analytics

Visit GitHub


This is an output of a large dataset related to international student enrollment at Centennial College. I went through this with MS Excel and mostly used the Pivot table.

Details

  • The 1'st bottom tap:

    It represents raw data including various attributes(columns)

  • The 2'nd bottom tap:

    It is named 'Confirmed(No duplicated)' and is the dataset handled by data cleaning (removed duplicated records and filled a blank cell with proper values)

  • The 3'rd bottom tap:

    It is named 'TOTAL_Recruit' is organized outcomes sorted by countries and school programs. It involves many meaningful results such as a subset of total students registered a course. Since it also includes the target number of students that management team expects in next semester, the data table helps determine how many more students will be enrolled in a particular program or from countries, or there are a large number of enrolled students, how much to reduce the number of target students for that program next semester

  • The 4'th and 5'th bottom tap:

    It is named each 'By School' and 'By Team' are literally a summary of the data sorted by school programs and teams (category of countries)
